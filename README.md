<p align="center">AutoPickupXP enhances the vanilla Minecraft experience system <br>
by introducing instant XP collection with smart routing, Mending support, and per-player toggles.<br><br>
It removes the need to chase XP orbs while keeping mechanics balanced and intuitive,<br>
making it perfect for survival servers, grinders, and performance-focused gameplay.<br><br>
Collect XP instantly, effortlessly!</p>

---

### <p align="center">Instant XP Pickup</p>
<p align="center">Automatically sends XP directly to players.<br>
Works for both block breaking and mob kills.</p><br>

### <p align="center">Smart Targeting</p>
<p align="center">XP is assigned to the correct player using location tracking.<br>
Optional nearest-player fallback for maximum compatibility.</p><br>

### <p align="center">Full Mending Support</p>
<p align="center">XP is intelligently used to repair items with Mending.<br>
Matches vanilla behavior while improving efficiency.</p><br>

### <p align="center">Player Toggle System</p>
<p align="center">Players can enable or disable the feature individually.<br>
State is saved and persists across restarts.</p><br>

### <p align="center">Performance Friendly</p>
<p align="center">Reduces entity clutter by removing XP orbs.<br>
Helps improve server performance in farms and grinders.</p>

---

### <p align="center">🧩 Requirements</p>
<p align="center">Paper / Spigot / Bukkit (1.20+)</p>

---

### <p align="center">⚙️ Installation</p>
<p align="center">1. Drop the plugin ".jar" into your plugins folder.</p>
<p align="center">2. Restart your server.</p>

---

### <p align="center">⚙️ Configuration Example</p>

```yaml
enabled: true
delay-ticks: 2
use-nearest-player: true

players: {}
```
